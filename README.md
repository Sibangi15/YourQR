# 📱 QR Code Scanner & Generator

A simple and elegant Flutter application that allows users to **generate and scan QR codes** in real-time.

## Live Link: 

https://qr-code-app-eta.vercel.app

## 🚀 Features

* 🔳 Generate QR codes from text or URLs
* 📷 Scan QR codes instantly
* 🎨 Clean and minimal UI
* ⚡ Fast and lightweight performance

## 🛠️ Tech Stack

* Flutter
* Dart
* pretty_qr_code package

## 📦 Future Improvements

* 📥 Download QR as image
* 📤 Share QR codes
* 🕘 QR scan history
* 🎨 Custom QR styles
